# Moonsweeper

A weekend project to play with my [Moonlander](https://www.zsa.io/moonlander) using the [ZSA API](https://github.com/zsa/kontroll).

## Usage

Start the Keymapp application and run API server.
```
cargo run --release --example wipe
```
